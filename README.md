# doslife

16-bit real mode implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life). Requires DOSBox or a real DOS computer.


![doslife](ASSETS/doslife.gif)

### Prerequisites
+ Turbo C++ (>= 3.0)
+ Turbo Assembler (>= 3.2)

### Building
```bash
MAKE.EXE
```
